# AI 词汇对应功能

## 功能介绍

本功能利用 AI 的强大语义理解能力，在翻译时直接返回原文词汇与译文词汇的对应关系，实现精准的词汇级高亮。

## 工作原理

### 1. Prompt 设计

在翻译请求中，要求 AI 同时返回词汇对应表：

```
【输出格式要求】
请按以下格式输出：

原文段落 | 中文翻译

[词汇对应]
英文单词1 | 对应中文1
英文单词2 | 对应中文2
...
[/词汇对应]
```

### 2. AI 返回示例

```
The network provides connectivity for fans. | 网络为粉丝提供连接。

[词汇对应]
network | 网络
provides | 提供
connectivity | 连接
fans | 粉丝
[/词汇对应]
```

### 3. 解析与存储

脚本解析 AI 返回的词汇对应表，建立映射关系：

```javascript
const wordMap = new Map();
wordMap.set('network', ['网络']);
wordMap.set('provides', ['提供']);
wordMap.set('connectivity', ['连接']);
wordMap.set('fans', ['粉丝']);
```

### 4. 高亮机制

**原文词汇悬停时：**
1. 整段译文获得浅黄色背景（段落级）
2. 根据 `wordMap` 查找对应的中文词汇
3. 译文中匹配的中文词汇获得深黄色背景（词汇级）

**译文词汇悬停时：**
1. 整段原文获得浅绿色背景（段落级）
2. 反向查找 `wordMap`，找到对应的英文单词
3. 原文中匹配的英文单词获得深绿色背景（词汇级）

## 优势

1. **语义准确**：AI 理解上下文，对应关系更准确
2. **处理一词多义**：AI 能根据语境选择正确的对应词汇
3. **支持意译**：即使译文与原文结构差异大，仍能建立对应关系
4. **可扩展性**：可以轻松添加更多对应类型（如短语、习语）

## 使用示例

### 示例 1：简单句子

```
原文：The cat sat on the mat.
译文：猫坐在垫子上。

[词汇对应]
cat | 猫
sat | 坐
mat | 垫子
```

悬停 "cat" → 译文 "猫" 高亮

### 示例 2：专业术语

```
原文：GPS jamming affects navigation systems.
译文：GPS干扰影响导航系统。

[词汇对应]
jamming | 干扰
navigation | 导航
systems | 系统
```

悬停 "jamming" → 译文 "干扰" 高亮
悬停 "GPS"（专业术语保留）→ 译文 "GPS" 高亮

### 示例 3：意译情况

```
原文：It's raining cats and dogs.
译文：下着倾盆大雨。

[词汇对应]
raining cats and dogs | 倾盆大雨
```

即使结构差异大，AI 仍能建立对应关系


## 限制

1. **依赖 AI 质量**：词汇对应的准确性取决于 AI 模型的理解能力
2. **额外 Token 消耗**：返回词汇对应表会增加 API 调用成本
3. **格式一致性**：需要 AI 严格按照约定格式返回

## 未来改进

1. **短语对应**：支持多词短语的对应关系
2. **多对多映射**：处理一个英文词对应多个中文词的情况
3. **置信度分数**：AI 返回对应关系的置信度
4. **用户反馈**：允许用户纠正对应关系，持续优化
